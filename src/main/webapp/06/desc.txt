세션
유저가 브라우저를 통해 접속하고 작업을 한뒤에 웹 브라우저를 닫아서 접속을 종료할 때까지를
하나의 세션이라고 한다.

세션 설정
유지 시간 설정
1. web.xml에서 session-config (분 단위) 로 설정하는 방법
2. JSP 파일에서 setMaxInactiveInterval()  (초 단위)

DTO 와 DAO

DTO (Data Transfer Object) 계층 사이에서 데이터를 교환하기 위해 생성되는 객체
별다른 로직이 없고 속성(멤벼 변수) 그 변수에 접근하기 위한 게터/세터 메서드만 가지고 있다.

DAO (Data Access Object)
DAO는 데이터 접근 객체 보통 JDBC를 통해 구현하고, 하나의 테이블에서 수행할 수 있는
CRUD를 전담한다.


    쿠키 VS 세션
            쿠키                              세션
저장 위치  클라이언트에 text로저장           웹 서버에 Object로 저장
보안      클라이언트에 저장 > 취약          서버에 저장 > 안전
자원/속도   서버 자원을 사용X 세션보다 빠름   서버 자원을 사용 > 쿠키보다 느림
용량      제한 O                          서버가 혀용하는 한 제한 X
유지 시간   쿠키 생성 시 설정                  web.xml에서 설정
        설정된 시간이 경과되면 무조건 삭제  설정된 시간 내라도 동작이 있다면 삭제되지 않음
