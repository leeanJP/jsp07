쿠키는 클라이언트의 상태정보를 유지하기 위한 기술
상태정보를 브라우저에 key value 형태로 저장했다가
다음 요청 시 저장된 쿠키를 전송 한다.

쿠키 3000개 만들수 있고
쿠키 하나의 최대크기는 4096Byte
하나의 호스트나 도메인에서 최대 50개 까지 만들수 있다.

쿠키의 속성
name : 쿠키를 구별하는 이름
문자열을 입력하는데 , ; space = 포함 X
value : 쿠키에 저장할 실제 데이터
domain : 쿠키를 적용할 도메인
path : 쿠키를 적용할 경로
특정 경로를 입력하면 하위경로도 포함 된다.
max age : 쿠키를 유지할 기간
age 설정 안하면 getAge -1 반환
